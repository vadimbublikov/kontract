# Kontract
Kontract is a Kotlin library for describing a domain using DSL. It allows you to declaratively define entities, their attributes, relationships, and business operations, creating a single source of knowledge about your system. A description in the form of code, instead of diagrams, allows LLM to better understand your project.  

---------------------

Kontract — это Kotlin-библиотека для описания предметной области с использованием DSL. Она позволяет декларативно определять сущности, их атрибуты, связи и бизнес-операции, создавая единый источник знаний о вашей системе. Описание в виде кода, вместо диаграмм, позволяет LLM лучше понимать ваш проект.
